# 📌 VisionGrid

VisionGrid is a Pinterest-inspired full-stack image gallery application where users can upload, organize, download, and save images in their wishlist. It features a clean, responsive UI with Dark/Light mode support.

## 🚀 Live Demo

🔗 https://vision-grid-one.vercel.app/

## ✨ Features

- 🔐 User Authentication
- 🖼️ Upload Images
- 📂 Image Categories
- ❤️ Wishlist
- 📥 Download Images
- 🗑️ Delete Images
- 🌙 Dark / Light Mode
- 📱 Fully Responsive Design

## 📌 Project Overview

VisionGrid was initially built as an image gallery project and later extended into a full-stack application. It allows authenticated users to upload, organize, manage, and download images while providing a modern, responsive user experience.

---

## 📋 Original Requirements

- Design an image gallery using HTML & CSS
- Add JavaScript navigation (Lightbox, Next/Previous)
- Add Hover Effects & Smooth Transitions
- Ensure Responsive Design
- Bonus: Image Categories
